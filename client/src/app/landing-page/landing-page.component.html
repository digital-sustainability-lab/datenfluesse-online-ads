Third Party Request Visualization

Motivation

  • Analyze and visualize web traffic by combining client-based browser automation (webXray) with modern data visualization tools.
  • Description of online advertisement ecosystem
  • Significance of online advertisement for websites propagating disinformation, data collection of online users for pre-targeting purposes in disinformation campaigns.
  • Highlight extent of third-party HTTP requests of popular websites in Switzerland and of alternative media websites. Education of citizens.

Methods

This website uses the following technologies:

  • Angular: A typescript framework used to build this website.
  • d3: A JavaScript library used for all the data visualizations.
  • webXray: A tool programmed in python that automatically collects traffic data in a chrome browser. The collected data is saved in a database.
  • Klazify: A webservice that gathers information on a given url (e.g. country or company).
